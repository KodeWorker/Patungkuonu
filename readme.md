## Description
This is my persional project aimed at building a custom 2D game engine.
Patungkuonu is the name of the highest mountain in my home country.

## Build Instructions
### Windows
```powershell
mkdir build
cd build
cmake -G"Visual Studio 17" -A"x64" ..
cmake --build . --config release
```
### Linux
```console
mkdir build
cd build
cmake ..
make
```
